#Editor Protocol
In this document, we're going to detail the protocol used by tools to communicate with a game instance.

# Concepts
## Endpoint
An endpoint is an url that offers a service related to a specific functionnality.

## Request
A request is a message transmitted by a client. Each request must receive a corresponding response from the server

## Response
A response is a message transmitted by the server to a client to indicate the status of a request
